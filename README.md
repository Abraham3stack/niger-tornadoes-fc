# Niger Tornadoes FC Website ⚽

A modern, responsive football club website built with React, Vite, Tailwind CSS, and shadcn/ui.

## 🌐 Live Demo


---

## 📌 Project Overview

This project is a professional football club website template customized for **Niger Tornadoes FC**, a Nigerian Premier Football League club.

It is designed to help football clubs:
- Showcase players and squad information
- Share news, match updates, and announcements
- Attract sponsors and increase visibility
- Provide a professional online presence

---

## 🚀 Features

### 🏠 Home
- Hero section with club branding
- Call-to-action buttons

### 📖 About Club
- Club history and identity
- Mission and vision

### 👥 Squad
- Player grid layout
- Player name, position, and jersey number

### 📰 News / Updates
- Match results
- Announcements
- Trials

### 📅 Fixtures & Results
- Upcoming matches
- Previous results

### 🖼️ Gallery
- Match and training photos

### 🤝 Sponsors
- Sponsor showcase section

### 📞 Contact
- Contact form
- Club information

### 📝 Join / Trials
- Player registration form

To reuse:
- Update club data (name, colors, content)
- Replace images
